# Fabrial Upgrades

*Roll: 1d8*

| Range | Result |
|---|---|
| 1 | **Amplified.** Attacks made with this fabrial gain an advantage.\ \ **Req.** Can only be applied to fabrials that have an attack. |
| 2 | **Reliable.** When making tests with this fabrial, ignore the first c you roll with it each scene. When you do, you gain no bonus from it.\ \ **Req.** Can only be applied to fabrials that require attack tests or other skill tests. |
| 3 | **Fine-Tuned.** You can spend O to recover 1d4 focus.\ \ **Req.** Can only be applied to fabrials that require attack tests or other skill tests. |
| 4 | **Efficient.** When you roll O while within 5 feet of the activated fabrial, you can spend it to make the fabrial regain a charge.\ \ **Req.** Can only be applied to fabrials that expend charges. |
| 5 | **Higher Capacity.** Increase the fabrial's maximum charges by 1.\ \ **Req.** Can only be applied to fabrials that expend charges. |
| 6 | **Long Ranged.** The fabrial's range is doubled.\ \ **Req.** Can only be applied to fabrials that have a range or a radius. |
| 7 | **Faster.** Increase the movement rate of the effect by 50 percent.\ \ **Req.** Can only be applied to fabrials that grant you a movement rate or cause you to move. |
| 8 | **Greater Damage.** Increase the damage dealt or healed by one die size (such as by increasing d4 to d6).\ \ **Req.** Can only be applied to fabrials that deal or heal damage. |
