**Activation:** 0

*Your mind quickly files and organizes data, and you swiftly act on that information while under pressure.*

Spend 2 focus to gain 2, which you can only use to make cognitive skill tests via the following actions: [[Actions/Basic/Use A Skill|Use a Skill]], [[Actions/Basic/Gain Advantage|Gain Advantage]], or an action from any Agent talent.

During a conversation or endeavor, at the GM’s discretion, you can instead use both 2 to make an additional contribution for the round.
