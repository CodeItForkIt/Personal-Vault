The metric system is a base 10 system that is the expected system of measurement within STEM. 

| Measurement | Milli-      | Centi-      | Deka-      | Basic  | Hecto-      | Kilo-      | Measurement Tool             |
| ----------- | ----------- | ----------- | ---------- | ------ | ----------- | ---------- | ---------------------------- |
| Length      | Millimeters | Centimeters | Dekameters | Meters | Hectometers | Kilometers | Metric Ruler                 |
| Volume      | Milliliters | Centiliters | Dekaliters | Liters | Hectoliters | Kiloliters | Beakers, Graduated Cylinders |
| Mass        | Milligram   | Centigram   | Dekagram   | Gram   | Hectogram   | Kilogram   | Scale                        |
|             | 0.001x      | 0.01x       | 0.1x       | 1x     | 100x        | 1000x      |                              |
